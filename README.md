Manage Requests Frontend

-Frontend được xây dựng bằng React và Vite.

-Ứng dụng kết nối với Backend Spring Boot thông qua API.

-Yêu cầu trước khi chạy

Cài đặt Node.js (khuyến nghị phiên bản 18 trở lên) Cài đặt npm (đi kèm với Node.js)

-Cách tải và chạy bằng file ZIP

Bước 1: Tải file ZIP

Vào trang GitHub của project.

Nhấn nút Code.

Chọn Download ZIP.

Giải nén file vừa tải về.

Bước 2: Mở project

Sau khi giải nén:

Mở thư mục project

Vào thư mục frontend

Nếu bên trong có thư mục vite-app thì vào tiếp vite-app bằng câu lệnh cd frontend/vite-app trong terminal

-Cài đặt thư viện

Mở Terminal trong đúng thư mục có package.json và chạy:

npm install

Lưu ý: Phải chạy lệnh này trong thư mục có file package.json.

-Chạy ứng dụng

Sau khi cài đặt xong, chạy:

npm run dev

-Mặc định ứng dụng sẽ chạy tại:

http://localhost:5173

-Lưu ý

Backend phải đang chạy trước khi sử dụng frontend.
